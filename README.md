> ### 컨셉

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;현대 의류 쇼핑몰

> ### 기능

* 사용자
  * 회원 가입 및 로그인
  * 아이디 찾기
  * 비밀번호 재설정

  * 메인 페이지
  * 문의글 작성
  * 상품 카트에 등록
  * 상품 상세보기 페이지
  * 카테고리별 상품 목록 페이지
  
* 관리자
  * 로그인
  * 회원 목록 조회
  * 문의사항 관리
  * 상품 등록
  * 상품 목록 조회, 관리
